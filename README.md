# 4h_tool

**obs tool for streaming video files using playlist**

allows you to create and manage playlists of video files that can be streamed through obs. supports various video formats and provides playlist controls for seamless streaming sessions.

## features

- playlist creation and management
- video file format support
- obs integration
- playlist controls (play, pause, next, previous)
- queue management
